Q1、APP可正常运行的操作系统版本号，最低是多少？


macOS 12.0

Q2、APP正确的安装姿势？


双击OC2PatchTool.app即可运行。


Q3、打不开“OC2PatchTool.app”，因为它来自身份不明的开发者


方案1.系统偏好设置>>安全与隐私>>允许安装未知来源


方案2.打开 Terminal 终端后 ，在命令提示后输入

sudo spctl --master-disable
